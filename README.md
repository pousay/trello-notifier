# 📡 Telegram Event Tracker (Trello → Telegram)

A production-ready, extensible Python service that tracks **Trello board events** and sends them as formatted messages to a **Telegram chat**.

Built with clean architecture, async Telegram bot (aiogram), and pluggable event sources.

---

## ✨ Features

- 📌 Tracks Trello board activity in real-time (polling-based)
- 🤖 Sends formatted updates to Telegram
- 🧱 Clean architecture (SOLID, OOP, extensible)
- 🔌 Pluggable event sources (Trello, GitHub, Jira, etc.)
- 🧾 Structured logging
- 🧠 Pydantic-based configuration & models
- ⚡ Async Telegram bot (aiogram v3)
